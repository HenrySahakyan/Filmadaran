
## ✨ Features

- 📺 **Browse & Discover** — Paginated TV show catalog via TVMaze API
- 🔍 **Live Search** — Debounced real-time search with automatic results
- ⭐ **Favorites** — Persistent local favorites with Room database
- 🔄 **Swipe Refresh** — Pull-to-refresh with smart state handling
- 🎭 **Details View** — Rich show details with animated transitions
- 🌐 **Localization** — Armenian, English, Russian

## 🛠 Tech Stack

| Layer | Technology |
|-------|-----------|
| **Language** | Kotlin |
| **Architecture** | Clean Architecture + MVVM |
| **DI** | Koin |
| **Networking** | Retrofit + OkHttp (10MB cache) |
| **Database** | Room (indexed queries) |
| **Reactive** | Kotlin Coroutines + StateFlow |
| **UI** | Material 3, Edge-to-Edge, ViewBinding |
| **Image Loading** | Glide (disk + memory caching) |
| **Logging** | Timber |
| **Optimization** | R8, resource shrinking |
